# apk release
# 1. Make sure everything is committed and pushed
git add .
git commit -m "your message"
git push

# Delete the tag locally and on GitHub
git tag -d v1.0.*
git push origin --delete v1.0.*

# 2. Tag it and push the tag — this triggers the CI build
git tag v1.0.2
git push origin v1.0.2


# Version naming
Just increment the tag each time:
v1.0.0  →  first public build
v1.0.1  →  small fix
v1.1.0  →  new feature
v2.0.0  →  big update
